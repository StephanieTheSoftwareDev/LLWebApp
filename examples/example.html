LegaLucy

This code is the prototype software for LegaLucy. Assumed json-server started on port 3000 
 terminal:> json-server examples/examples.json

Data Model

Guides are content-based widgets that allow users to explore or a topic or narrow down a process to uncover which legal content is appropriate.
  • Guide (similar to a presentattion)
    • Node (questions, videos, or text,images content, broken down into approx. paragraph junks )
      • choice. the choices are answers or next buttons

    Example JSON

    

 "guide": [

      "id": 1,
      "name": "guide 1"

"node": [

    Example Queries

    Get Guide titles for drop down

    This returns full object; A middleware route is needed; Get all properties amd just build for title
    http://localhost:3000/guide

    Get a Specific Guide

    This returns the entire guide with nodes. Use this for when editing an existing guide
    http://localhost:3000/guide/1?_embed=node

    Get all node titles for a Guide This needs middleware to filter out the extra elements.

    Middleware

    Middleware will help streamline the query process and handle authorization, logging and other functionality.
